## 🔄 Omni Scraper Update Report
*2026-03-24T19:58:38Z*

### 📋 New Changelog Entries (10)
- `2026-03-12` Added support for custom model identifiers in OpenAI-compatible endpoints, allowing you to bring your own models (BYOM) for providers like Ollama and Azure OpenAI. See the AI settings documentation for more information.
- `2026-03-12` Cross-filtering is now generally available. You can use cross-filtering to filter data across multiple dashboard tiles by clicking on values in your visualizations. See the cross-filtering documentation for more information.
- `2026-03-12` You can now ask the AI chat assistant to generate map visualizations for geographic questions. When you ask questions like "show me orders by state on a map," the AI will create region maps (choropleths) or point maps as appropriate for your data.
- `2026-03-11` Added onlySharedWithMe as a new filter option to the List folders API to retrieve only folders that have been explicitly shared with you via folder permits, excluding folders you own.
- `2026-03-11` Added support for generic embed integrations like FullStory and LogRocket. You can now configure integration credentials in Settings > Embed > Integrations .
- `2026-03-11` Renamed AI workflows to "skills" throughout Omni. Existing model files using workflows: will continue to work without changes.
- `2026-03-10` Added onlySharedWithMe as a new filter option to the List documents API to retrieve only documents that have been explicitly shared with you via documents permits, excluding documents you own.
- `2026-03-10` Added a button toggle control type for single selection string filters, allowing you to display filter options as clickable buttons instead of a dropdown. Available for Advanced Dashboard layouts only.
- `2026-03-09` Added a confirmation dialog when disabling personal access tokens . You must now confirm the action before a token is disabled, helping prevent accidental disablement.
- `2026-03-09` Added support for creating multiple embed secrets in an Omni instance.
